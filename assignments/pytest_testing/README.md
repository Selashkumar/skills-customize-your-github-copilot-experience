# 📘 Assignment: Testing Python Programs with pytest

## 🎯 Objective

Learn how to write automated unit tests in Python using `pytest` and ensure small functions behave correctly.

## 📝 Tasks

### 🛠️ Write testable Python functions

#### Description

Create a Python module with several utility functions that can be tested automatically.

#### Requirements
Completed program should:

- Define at least three functions in `starter-code.py`
- Include functions for string processing, arithmetic, and list handling
- Keep the functions pure and easy to test (no user input inside the functions)
- Use clear function names and docstrings

### 🛠️ Create pytest test cases

#### Description

Write tests with `pytest` to verify your functions return the expected outputs and handle edge cases.

#### Requirements
Completed program should:

- Include a `tests/test_starter_code.py` file with test cases for each function
- Use `assert` statements to validate correct behavior
- Test both normal values and edge cases where appropriate
- Run successfully with the `pytest` command
